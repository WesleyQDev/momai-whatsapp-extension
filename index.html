Requisitos do Sistema

Tudo o que você precisa para rodar a MomAI com performance.

MIN

Mínimo

  • Windows 10 ou 11 (64-bit)
  • 8GB RAM
  • CPU 4 Cores (Intel i5 / Ryzen 5)
  • Python 3.10+ (Recomendado para uv/Sandbox)
  • API Key (OpenAI, Anthropic) OU Local LLM

Recomendado

  • Windows 11 (64-bit)
  • 16GB+ RAM (Ideal para LLMs locais)
  • CPU 8 Cores + GPU NVIDIA (RTX 3060+)
  • Ollama ou LM Studio instalado
MIN

Mínimo

  • Ubuntu 22.04 / Fedora 38+
  • 8GB RAM
  • Python 3.10+ (Recomendado para uv/Sandbox)
  • API Key (OpenAI, Anthropic) OU Local LLM

Recomendado

  • Distro Moderna com Wayland/X11
  • 16GB+ RAM (32GB para modelos grandes)
  • NVIDIA GPU com suporte a CUDA
MIN

Mínimo

  • macOS Sonoma 14.0+
  • 8GB RAM (Unified Memory)
  • Apple Silicon (M1/M2/M3) ou Intel
  • API Key (OpenAI, Anthropic) OU Local LLM

Recomendado

  • macOS Sequoia 15.0+
  • 16GB+ Unified Memory
  • Apple Silicon M2 Pro ou superior
