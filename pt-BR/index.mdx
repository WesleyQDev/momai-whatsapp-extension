## Comunidade BR

Criado no Brasil, a MomAI é um projeto de código aberto sob a licença MIT, desenvolvido como uma plataforma que integra diversas soluções, garantindo que o usuário não fique limitado a uma única opção.

### Resumo rápido das tecnologias (Nodejs + Python)

- **NodeJS (Electron + Vite + React)** - inicializador, Interfaces gráficas, gerenciamento de atualização da aplicação.
- **Langraph/Langchain** - Orquestração dos agentes
- **FastAPI** - Conexão entre o Electron e o núcleo da Assistente Python, Usando WebSockets e Api
- **llama.cpp** - Execução de modelos locais LLM
- **Kokoro/ rany2/edge-tts** - Text to speak
- **Whisper** - Speak to Text
- **APScheduler** - Agendamentos de funções Python
- **DuckDuckGO** - Ferramenta de busca na internet
- **Mintlify** - Documentação

### Ferramentas

- **Git** - Gerenciador de versões
- **GitHub** - CI/CD, releases

**Python**

- uv (Gerenciador de pacotes)
- Ruff (Lint)
- Tasker (Python)
- Pytest (Testes)

**Node.js**

- nvm (Gerenciador de versões Node)
- pnpm (Gerenciador de pacotes)
- ElectronBuilder / ElectronForge (Empacotador electron)
