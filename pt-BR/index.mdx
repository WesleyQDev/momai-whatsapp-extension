## Comunidade BR

Criado no Brasil, MomAI é de código aberto o projeto usa diversas soluções para que você não fique preso a uma única possibilidade. Você pode contribuir com o projeto de diversas formas:

1. Criando Issues
2. Realizando melhorias no front-end ou no núcleo back-end
3. Desenvolvendo e publicando extensões para a comunidade
