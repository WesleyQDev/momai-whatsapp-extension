## Comunidade BR

Criado no Brasil, MomAI é de código aberto, o projeto usa diversas soluções para que o usuario não fique preso a uma única possibilidade.

### Resumo rapido das techs

- Langraph/Langchain - Orquestração dos agentes
- Electron.js + Vite + ReactJS- Interface gráfica
- FastAPI - Coneção entre o Electron.js e o núcleo da Assistente Python, Usando Websokets e Api
- Kokoro - Text to speak
- Whisper - Speak to Text
- ApsScheduler - Agendamentos de funções Python
- DuckDuckGo, Brave Search e Tavily - Ferramentas de buscas na internet
- Mintlify - Documentação

### Ferramentas

- Git

**Python**

- uv (Gerenciador de pacotes)
- Ruff (Lint)
- Tasker (Python)
- Pytest (Testes)

**Node.js**

- nvm (Gerenciador de versões Node)
- pnpm (Gerenciador de pacotes)
- ElectronBuilder / ElectronForge (Empacotador electron)
