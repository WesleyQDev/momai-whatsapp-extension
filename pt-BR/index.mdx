## Comunidade BR

Criado no Brasil, o MoMAI é um projeto de código aberto sob a licença MIT. Desenvolvido sob uma arquitetura de monorepo modular, ele integra diversas soluções em um único ecossistema coeso, garantindo que o usuário tenha o poder de utilizar a plataforma completa ou apenas os módulos que desejar. Essa estrutura permite que o projeto evolua de forma ágil e colaborativa, sem que você fique limitado a uma única solução fechada.

### Resumo rápido das tecnologias (Nodejs + Python)

- **NodeJS (Electron + Vite + React)** - inicializador, Interfaces gráficas, gerenciamento de atualização da aplicação.
- **Langraph/Langchain** - Orquestração da IA 
- **FastAPI** - Conexão entre o Electron e o núcleo da Assistente Python, Usando WebSockets e Api
- **llama.cpp** - Execução de LMM locais (Suporta CPU, placas antigas AMD e GPUS da NVIDIA)
- **Kokoro/ rany2/ edge-tts** - Bibliotecas usadas para transcrever o que o usuario esta falando (TTS)
- **Whisper** - Biblioteca usada para a voz da MomAI (STT)
- **APScheduler** - Agendamentos de funções Python
- **Mintlify** - Documentação
