Você já deixou um bot, uma API ou um script rodando em segundo plano enquanto jogava, apenas para notar que o jogo começou a travar? Ou esqueceu processos consumindo memória silenciosamente até que seu PC ficou lento?

O **FortScript** resolve isso automaticamente. Ele pausa seus scripts quando você abre um jogo ou aplicativo que exige muitos recursos e os retoma quando você os fecha. Simples assim.

## Como funciona?

1. **Definição**: Você define quais scripts deseja gerenciar (Bots Python, projetos Node.js, executáveis, etc.).
2. **Gatilhos**: Você define quais aplicativos são "pesados" (jogos, editores de vídeo, etc.).
3. **Automação**: O FortScript monitora o sistema e faz o resto: pausa quando necessário e retoma quando possível.

### Eventos de Callback (Opcional)

Você pode configurar funções que serão executadas automaticamente:
- **on_pause**: Executada quando os scripts são pausados (ex: enviar notificação, salvar estado).
- **on_resume**: Executada quando os scripts são retomados (ex: reconectar serviços, registrar log).

## Como Usar

### Opção 1: Configuração Básica (Apenas YAML)
A maneira mais simples, carregando as configurações de um arquivo `fortscript.yaml`.

```python
from fortscript import FortScript

# Carrega as configurações de fortscript.yaml
app = FortScript()
app.run()
```

### Opção 2: Com Callbacks de Evento
Execute funções personalizadas nos momentos de pausa e retomada.

```python
from fortscript import FortScript, Callbacks

def ao_pausar():
    print("🎮 Modo Jogo ativo! Scripts pausados.")

def ao_retomar():
    print("💻 De volta ao trabalho! Scripts retomados.")

callbacks = Callbacks(
    on_pause=ao_pausar,
    on_resume=ao_retomar,
)

app = FortScript(
    config_path="fortscript.yaml",
    callbacks=callbacks,
)
app.run()
```

### Opção 3: Configuração Completa (Python Dinâmico)
Ideal para manter o código organizado sem depender exclusivamente de arquivos externos.

```python
from fortscript import FortScript, RamConfig, Callbacks

# 1. Defina seus projetos para gerenciar
meus_projetos = [
    {"name": "Discord Bot", "path": "./bot/main.py"},
    {"name": "Express API", "path": "./api/package.json"},
]

# 2. Defina os processos pesados
processos_pesados = [
    {"name": "GTA V", "process": "gta5"},
    {"name": "Chrome", "process": "chrome"},
]

# 3. Inicialize o FortScript
app = FortScript(
    projects=meus_projetos,
    heavy_process=processos_pesados,
    ram_config=RamConfig(threshold=90, safe=80),
    callbacks=Callbacks(
        on_pause=lambda: print("Pausado!"),
        on_resume=lambda: print("Retomado!")
    ),
    log_level="DEBUG",
)
app.run()
```

### Opção 4: Via CLI (Terminal)
Ideal para testes rápidos.
```bash

```
