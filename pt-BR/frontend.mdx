O frontend é responsável pela interface gráfica e por gerenciar a comunicação com o núcleo Python via FastAPI.

## Pré-requisitos

<Info>
  **Tecnologias principais:**
  - **Node.js 19+**
  - **pnpm** (Gerenciador de pacotes recomendado)
  - **Electron Forge** (Empacotador)
</Info>

## Passos para Instalação

### 1. Instalar dependências
Navegue até a pasta do projeto e execute:
```bash
pnpm install
```

### 2. Executar em modo de desenvolvimento
Isso iniciará o Vite para o hot-reload do React e abrirá a janela do Electron.
```bash
pnpm run dev
```

### 3. Build e Empacotamento
Para gerar o executável final para o seu sistema operacional:
```bash
pnpm run make
```

## Estrutura de Comunicação

O frontend se conecta ao backend (Python) através de uma porta local definida nas configurações. Ele utiliza **WebSockets** para receber atualizações em tempo real (como o texto que está sendo transcrito pela voz) e **APIs REST** para comandos diretos.
