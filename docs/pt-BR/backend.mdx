Recomendamos configurar o backend e o frontend para garantir que a comunicação via WebSockets funcione corretamente durante os testes.

## Pré-requisitos

<Info>
  **Verifique as versões instaladas:**
  - **Python 3.12** ou superior (Recomendamos o uso do `uv` para gestão)
  - **Node.js 19** ou superior (pnpm como gerenciador de pacotes)
  - **Git** para clonar os módulos e extensões
</Info>

## Passos para Instalação

### 1. Clonar o projeto
```bash
git clone https://github.com/WesleyQDev/momai
cd momai
```

### 2. Configurar o Python (Núcleo)
Nós usamos o `uv` pela sua velocidade e facilidade com ambientes virtuais.
```bash
# Criar ambiente virtual e instalar dependências
uv sync
```

Se você preferir usar pip tradicional:
```bash
python -m venv .venv
source .venv/bin/activate  # No Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

### 3. Executar o Servidor FastAPI
O backend atua como um servidor que aguarda as conexões do Electron.
```bash
# Usando o tasker ou comando direto
python main.py
```

## Ferramentas de Desenvolvimento

- **Ruff**: Usado para manter o código limpo e padronizado.
- **Pytest**: Execute `pytest` para garantir que as alterações não quebraram o núcleo.
- **Tasker**: Atalhos para tarefas comuns de desenvolvimento.
