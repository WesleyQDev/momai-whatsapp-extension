Acompanhe a evolução da MomAI desde o início do projeto.

## 2026

<Update label="Janeiro 2026">
  ### Roteamento Inteligente e Tool RAG
  Lançamento das funcionalidades que definem a arquitetura atual da MomAI:

**Novidades:**

- **Semantic Router** — Roteamento baseado em embeddings usando LanceDB. Identifica intenções em milissegundos sem precisar do LLM.
- **Tool RAG** — Carregamento dinâmico de ferramentas. Permite escalar para centenas de extensões sem impacto na performance.
- **Wake Word Local** — Detecção da palavra "Sistema" 100% offline via Vosk.
- **Streaming TTS** — Resposta de voz em tempo real, falando enquanto ainda processa.

**Melhorias:**

- Redução de latência em comandos simples de ~2s para ~200ms
- Uso de RAM reduzido em 30% com lazy loading de extensões
</Update>

## 2025

<Update label="Dezembro 2025">
  ### Nascimento da Identidade
  A MomAI ganha forma e identidade visual:

**Novidades:**

- **Identidade visual oficial** — Logo, cores e design system definidos
- **Arquitetura híbrida** — Consolidação do modelo Electron + Python
- **Motor LLM local** — Integração com llama.cpp via llama-cpp-python
- **Suporte multi-LLM** — Fallback para Groq e Gemini quando necessário
- **Primeiras ferramentas** — Automações básicas de sistema operacional

**Stack definida:**

- Backend: FastAPI + LangGraph + SQLite + LanceDB
- Frontend: Electron + React 19 + Tailwind CSS
- Modelo padrão: Qwen3-4B-Instruct (GGUF)
</Update>

<Update label="06 de Dezembro 2025">
  ### Criação do FortScript
  Nascimento do projeto FortScript como solução para gerenciamento de recursos:

**Problema:** MomAI consome recursos significativos. Quando o usuário joga ou edita vídeos, precisa de toda performance disponível.

**Solução:** FortScript monitora processos e permite suspender módulos pesados automaticamente quando aplicações específicas são detectadas.

**Repositório:** [github.com/WesleyQDev/fortscript](https://github.com/WesleyQDev/fortscript)
</Update>

<Update label="18 de Novembro 2025">
