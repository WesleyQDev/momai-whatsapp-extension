A interface da MomAI foi desenhada para ser leve, moderna e, acima de tudo, dinâmica. Ela não apenas exibe texto, mas reage a comandos do backend para abrir interfaces gráficas e monitorar o sistema.

## Stack Tecnológica
- **React 19**: Aproveitando as últimas melhorias de performance e hooks.
- **Tailwind CSS**: Estilização baseada em variáveis globais para suporte a temas.
- **Lucide & Heroicons**: Conjunto unificado de ícones.
- **React Router 7**: Gerenciamento de rotas para views principais e de extensões.

## Comunicação em Tempo Real
A maior parte da inteligência de UI vem do **WebSocket** (`ws://localhost:8000/ws`).

### Eventos do Sistema
- `extensions_sync`: Enviado pelo backend ao conectar, sincroniza os ícones da barra lateral.
- `graph_open`: Aciona o `GraphInterface` para exibir componentes visuais (Gráficos, Forms).
- `resource_usage`: Telemetria de CPU/RAM/VRAM exibida no `ResourceFooter`.

## Componentes Dinâmicos

### LateralBar.tsx
Este componente é inteligente. Além das rotas fixas (`/`, `/reminders`), ele mapeia o array de extensões ativas. Se uma extensão declarar `sidebar: true`, o ícone aparecerá automaticamente aqui.

### GraphInterface.tsx
É o "palco" para interações complexas. Ele pode ser exibido no centro (modal) ou na lateral.
- **UI Schema**: Aceita um objeto JSON que define botões e ações, permitindo que o backend controle a UI sem precisar de código JS novo no frontend.

## Gerenciamento de Estado
Utilizamos Hooks customizados para isolar a lógica de negócio:
- `useChat`: Gerencia o histórico, o streaming de tokens e o estado do Grafo.
- `useStatus`: Monitora a saúde do backend e gerencia a troca de modelos de IA.

## Como rodar em desenvolvimento
```bash
# Na raiz do projeto
pnpm run dev
```
Isso iniciará o Vite e o Electron. O HMR (Hot Module Replacement) está ativo, então as mudanças no React refletirão instantaneamente.
