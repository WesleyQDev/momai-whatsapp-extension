O frontend é a janela onde você interage com a MomAI. Ele é construído com Electron, React e Vite.

<Info>
  O frontend precisa do backend rodando para funcionar. Se você ainda não configurou, veja o [guia do backend](/pt-BR/backend) primeiro.
</Info>

## Iniciando em modo de desenvolvimento

Se você já rodou `pnpm install` na raiz do projeto, as dependências já estão instaladas.

```bash
npm run dev:desktop
```

Isso vai:
1. Iniciar o servidor Vite (para hot-reload durante desenvolvimento)
2. Abrir a janela do Electron automaticamente

## Gerando um executável

Se você quiser usar o MomAI sem precisar de terminal:

```bash
npm run build
```

O instalador será gerado na pasta `apps/desktop/dist`.

## Problemas comuns

<AccordionGroup>
  <Accordion title="A tela está em branco">
    Verifique se o backend está rodando (`npm run dev:python`). O frontend tenta conectar em `localhost:8000`. Sem o servidor, o chat não funciona.
  </Accordion>
  <Accordion title="Erro com módulos nativos (node-gyp)">
    No Windows, você precisa das ferramentas de build do C++. Instale via Visual Studio Build Tools ou rode:
    ```bash
    npm install --global windows-build-tools
    ```
  </Accordion>
</AccordionGroup>
