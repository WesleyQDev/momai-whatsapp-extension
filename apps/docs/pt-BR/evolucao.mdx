## Contexto histórico / Changelog

<Steps>
<Step title="Janeiro de 2026: Nascimento da Identidade e Core">
  A MomAI ganha sua identidade visual oficial. No âmbito técnico, consolidamos a arquitetura híbrida Electron + Python, estabelecendo o processamento local via Qwen3-4B-GGUF (rodando nativamente através do llama-cpp-python) para garantir total soberania de dados. Integramos suporte multi-LLM com Groq e Gemini, e implementamos as primeiras ferramentas (tools) de automação, permitindo que a IA interaja diretamente com o sistema operacional.
</Step>
  <Step title="06 de dezembro de 2025">
    Criação do repositório fortscript. MomAI exige muitos recursos computacionais, não sendo viável manter alguns módulos ativos quando o usuário realiza tarefas pesadas como jogar ou editar vídeos. Fortscript é um projeto que atualmente é independente. MomAI usa essa biblioteca Python internamente para encerrar alguns módulos quando o usuário estiver realizando tarefas pesadas, ou quando o computador estiver com o consumo de RAM muito alto.

    https://github.com/WesleyQDev/fortscript

  </Step>
  <Step title="18 de novembro de 2025">
    A documentação do projeto foi migrada para a plataforma da Mintlify.
  </Step>
  <Step title="15 de novembro de 2025">
    Criação da documentação do projeto (a documentação começou a ser feito antes do próprio código), inicialmente desenvolvida em Obsidian.
  </Step>
    <Step title="Agosto de 2025">
    Pesquisa e elaboração de pequenos projetos, cada um testando algo diferente que uma assistente em um computador pode fazer.
  </Step>
  <Step title="Junho de 2025" iconType="solid">
    Projeto pessoal: Emi, um bot pessoal low code usando n8n para diversas automatizações.
  </Step>
</Steps>
