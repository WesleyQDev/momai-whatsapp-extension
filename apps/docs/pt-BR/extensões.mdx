## A filosofia

O MomAI foi construído para ser **incompleto de propósito**.

O núcleo faz o básico: conversar com você, manter contexto, conectar-se a modelos de IA. Mas o poder real vem das extensões que a comunidade cria.

Pense assim: o MomAI é o palco. As extensões são os artistas.

## O que uma extensão pode fazer?

Praticamente qualquer coisa:

- **Novos Agentes**: Um agente que organiza seus downloads. Outro que monitora preços.
- **Conectores**: Integrar com Notion, Obsidian, Google Calendar, Spotify.
- **Automações**: Responder emails, mover arquivos, controlar dispositivos IoT.
- **Interfaces**: Criar widgets visuais dentro do chat.

## Status atual

<Warning>
  O sistema de extensões ainda está em desenvolvimento. A arquitetura está sendo desenhada para ser simples de usar, mas poderosa o suficiente para casos complexos.
</Warning>

Quando estiver pronto, você poderá:

1. Baixar extensões da comunidade com um comando
2. Criar suas próprias extensões usando Python (para lógica) e React (para UI)
3. Compartilhar suas criações via um registro público

## Como contribuir agora?

Mesmo em desenvolvimento, você pode ajudar:

- **Testar o núcleo** e reportar bugs
- **Sugerir ideias** de extensões que você gostaria de ver
- **Acompanhar o progresso** no [repositório do GitHub](https://github.com/WesleyQDev/momai)
