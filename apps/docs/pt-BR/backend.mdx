O backend é onde a "inteligência" do MomAI vive. É um servidor Python (FastAPI) que recebe suas mensagens, processa com um modelo de IA e devolve a resposta.

## O que você vai precisar

- **Python 3.12** ou superior
- **Node.js 22** ou superior
- **pnpm** para gerenciar pacotes

## Passo a passo

### 1. Clone o repositório

```bash
git clone https://github.com/WesleyQDev/momai
cd momai
```

### 2. Instale as dependências

Na raiz do projeto, rode:

```bash
pnpm install
```

Isso cuida de tudo: dependências do Node (para o frontend) e do Python (para o backend).

### 3. Configure o modelo de IA

O MomAI é flexível quanto ao "cérebro" que você quer usar. Você pode:

**Usar um modelo local (offline, gratuito):**
Não precisa fazer nada. Na primeira execução, o sistema baixa automaticamente o modelo `Qwen3-4B` (~3GB).

**Usar uma API externa (Gemini, OpenAI, etc):**
Crie um arquivo `.env` na pasta `apps/core`:

```bash
# apps/core/.env
GEMINI_API_KEY="sua-chave-aqui"
```

### 4. Inicie o servidor

```bash
npm run dev:python
```

Quando você ver `Uvicorn running on http://127.0.0.1:8000`, está pronto.

<Note>
  O download do modelo local acontece apenas na primeira vez. Depois disso, a inicialização é rápida.
</Note>

## Verificando se funcionou

Acesse `http://127.0.0.1:8000/status` no navegador. Você deve ver algo como:

```json
{
  "status": "online",
  "version": "V0.1.0",
  "model": "local"
}
```
