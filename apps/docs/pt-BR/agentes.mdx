A MomAI não é um chatbot simples; ela é um ecossistema de agentes coordenados. Utilizamos o **LangGraph** para criar fluxos de trabalho complexos, cíclicos e resilientes.

## O Fluxo de Decisão

Quando você envia uma mensagem, ela passa por várias etapas de filtragem e decisão:

### 1. Roteador Semântico (LanceDB)
O primeiro passo é totalmente local. Usamos o **LanceDB** para comparar sua mensagem com uma base de dados de intenções pré-indexadas.
- **Vantagem:** Se você diz "Que horas são?", o sistema identifica instantaneamente que deve chamar o `SystemAgent` sem precisar gastar tokens em um LLM na nuvem.
- **Confiança:** Se a similaridade for baixa, o pedido é escalado para o Orquestrador Estratégico.

### 2. Orquestrador Estratégico
Se o roteador semântico não tiver certeza, o LLM principal analisa o pedido e decide qual especialista é o mais adequado. Ele pode escolher entre:

- **SearchAgent:** Especialista em buscas na web e síntese de informações.
- **SystemAgent:** Tem acesso ao sistema operacional (arquivos, processos, hardware).
- **SchedulerAgent:** Cuida da sua agenda e lembretes proativos.
- **InterfaceAgent:** Responsável por mudar o comportamento da UI ou exibir gráficos.
- **Responder:** O modo de chat geral para conversas informais.

### 3. Tool RAG (Retrieval Augmented Tools)
Diferente de outros assistentes que tentam enviar todas as funções disponíveis para o LLM, a MomAI usa **Tool RAG**. 
- O sistema busca no banco vetorial as ferramentas mais relevantes para o contexto atual.
- Apenas o "set" necessário de ferramentas é vinculado ao agente.
- Isso permite que a MomAI tenha milhares de ferramentas disponíveis sem "alucinar" ou perder performance.

## Agentes Especialistas

| Agente | Responsabilidade | Ferramentas Típicas |
|--------|------------------|-------------------|
| **System** | Controle do PC | `get_system_stats`, `manage_files`, `volume_control` |
| **Search** | Informação externa | `ddg_search`, `browser_control`, `weather_api` |
| **Scheduler** | Tempo e Agenda | `set_reminder`, `check_calendar`, `alarms` |
| **Interface** | Visualização | `show_graph`, `change_theme`, `open_settings` |

## Memória Persistente
Utilizamos o `AsyncSqliteSaver` do LangGraph para manter o estado da conversa. Isso significa que a MomAI lembra não apenas do texto, mas do progresso de tarefas complexas que envolvem múltiplos passos, mesmo que você feche a aplicação.
