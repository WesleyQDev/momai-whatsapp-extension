Identificar o nome real do processo de um jogo nem sempre é fácil (ex: o processo do Fortnite pode ser `fortniteclient-win64-shipping`). 

Para facilitar sua vida, o FortScript inclui um módulo chamado `games` com uma lista gigante de jogos e aplicativos populares já mapeados.

## Como usar a lista integrada

Em vez de preencher manualmente o `heavy_processes` no YAML, você pode importar e usar a lista `GAMES` diretamente no seu script Python.

```python
from fortscript import FortScript
from fortscript.games import GAMES

# GAMES contém centenas de entradas para:
# FPS, MOBAs, RPGs, Simuladores, Softwares de Edição, etc.

app = FortScript(heavy_process=GAMES)
app.run()
```

## Filtrando ou Combinando Jogos

Você pode combinar a lista padrão com seus próprios processos personalizados ou filtrar apenas o que deseja:

```python
from fortscript import FortScript
from fortscript.games import GAMES

meus_custom = [
    {"name": "Meu App Pesado", "process": "myapp"}
]

# Unindo a lista padrão com a personalizada
todos_pesados = GAMES + meus_custom

app = FortScript(heavy_process=todos_pesados)
app.run()
```
